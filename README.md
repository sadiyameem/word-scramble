# Word Scramble Game

# Description
- Word scramble is a game where you have to guess a word made from mixed-up letters within a time limit

# Features
- Start button
- Refresh button
- Check button
- A section to type the word

# How I made this
- HTML
- CSS
- JavaScript

# Motivation
- I wanted to make something quick and fun

# Challenges/Learned
- A challenge was to make the words not appear multiple times
- I learned how to use event listener to make buttons respond to clicks and run like starting the game
